## Component Specifications

### Colors (Tailwind Classes)
| Element | Light Mode | Dark Mode (Future) |
|---------|------------|-------------------|
| Primary | `blue-600` | `blue-400` |
| Background | `gray-50` | `gray-900` |
| User Bubble | `blue-600` | `blue-500` |
| Assistant Bubble | `gray-100` | `gray-800` |
| Text Primary | `gray-900` | `gray-100` |
| Text Secondary | `gray-500` | `gray-400` |
| Border | `gray-200` | `gray-700` |
| Error | `red-500` | `red-400` |
| Success | `green-500` | `green-400` |

### Role Badge Colors
| Role | Background | Text |
|------|------------|------|
| Admin | `red-100` | `red-800` |
| SuperUser | `purple-100` | `purple-800` |
| User | `blue-100` | `blue-800` |

### Category Badge Colors
| Type | Background | Text |
|------|------------|------|
| Global | `green-100` | `green-800` |
| Specific | `blue-100` | `blue-800` |

### Status Badge Colors
| Status | Background | Text |
|--------|------------|------|
| Active | `green-100` | `green-800` |
| Paused | `yellow-100` | `yellow-800` |
| Processing | `blue-100` | `blue-800` |
| Error | `red-100` | `red-800` |

### Spacing
- Sidebar width: `280px` (desktop)
- Chat padding: `16px`
- Message gap: `16px`
- Border radius: `8px` (cards), `16px` (bubbles)

### Typography
- Headings: `font-semibold`
- Body: `text-sm` to `text-base`
- Timestamps: `text-xs text-gray-500`
