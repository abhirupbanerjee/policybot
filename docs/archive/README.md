# Archived Documentation

This directory contains historical implementation documentation from the Policy Bot project.

## Why These Are Archived

These documents were created during specific implementation phases and provide detailed records of changes made at particular points in time. While they contain valuable historical context, they are no longer actively maintained or necessary for current operations.

## Contents

### Formatting Enhancements (December 2025)
- **FORMATTING_IMPROVEMENTS.md** - Comprehensive changes to system prompt for better response formatting
- **FORMATTING_QUICK_REFERENCE.md** - User guide for the formatting changes
- **SYNTAX_ALIGNMENT_VERIFICATION.md** - Technical verification that prompt syntax matches UI rendering
- **SYSTEM_PROMPT_ENHANCEMENT.md** - Documentation of system prompt updates

### Phase 1 Implementation (December 2025)
- **PHASE1_IMPLEMENTATION_SUMMARY.md** - Summary of terminology changes from "policy" to "knowledge base"
- **PHASE1_MIGRATION_GUIDE.md** - Migration guide for the Phase 1 changes

## Should You Read These?

**For Current Users:** No, use the current documentation in the parent [docs/](../) directory.

- What trade-offs were considered
